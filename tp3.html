1.1- ¿Qué significa CSS?

CSS significa Cascading Stylesheets es una tecnologia que se aprende despues de html.

1.2- ¿Para qué sirve el CSS? ¿Qué programa se utiliza para interpretar el CSS?

CSS sirve para darle estilo y un posicionamiento visualmente adecuado a la estructura definida por html. Para interpretarlo se usa el programa html

1.3- ¿A qué se llama regla de CSS?

se le llama regla de CSS a una propiedad o valor que se aplicara a algunas etiquetas segun se necesiten.

1.4- ¿Cuál es la anatomía de la regla de CSS?

Selector
selecciona aquellos elementos para dar estilo.

Declaracion
aplica las propiedades que les quieras dar al elemento seleccionado en el Selector.

Propiedades
es lo que te permite darle estilo a un elemento html, las propiedades se ubican a la izquierda.

Valor de la Propiedad
los valores se ubican a la derecha. cada propiedad tiene sus valores definidos.

1.5- ¿Qué es un selector en CSS?

El Selector sirve para seleccionar uno o varios elementos para posteriormente darles un estilo en la Declaracion.

1.6- Identificar y definir los selectores: de ID, de clases, universal, de etiquetas, descendente.

Selector Universal
Permite aplicarle estilos a todos los elementos html de un documento, se indica mediante el "*".

Selector por Etiquetas
Se le aplica a una etiqueta, se puede identificar sin utilizar "<>"

Selector Descendente
Este selector selecciona un elemento que se encuentra dentro de otro del mismo tipo, estan formados por dos o mas selectores separados.

Selector de Clases
Estos selectores empiezan por un punto (.), se usa class para definir el uso de los selectores de clases

Selector de Identidad
es un selector igual al Selector de Clases pero solo se aplica a un unico elemento html, esto debido a que el atributo id identifica a un elemento por su nombre

1.7- Explicar brevemente utilizando una lista ordenada: ¿cómo funciona el CSS?

      Seleccion de elementos
    • CSS permite seleccionar elementos html utilizando los llamados Selectores.

      Propiedades y Valores
    • Una vez seleccionado el elemento, se pueden aplicar propiedades y valores para cambiar el estilo.

      Reglas CSS
    • Una regla CSS consta de un selector y un bloque de declaraciones que contiene las propiedades y valores.

      Cascada y Herencia
    • El CSS sigue un principio de cascada, lo que significa que las reglas se aplican en un orden específico y pueden ser reemplazadas por reglas posteriores.
      Además, el CSS también permite la herencia, lo que significa que las propiedades se pueden heredar de elementos padre a elementos hijos.

      Hoja de estilos externa
    • El CSS también se puede utilizar en hojas de estilos externas, que son archivos separados que contienen todas las reglas CSS para un documento HTML.

1.8- ¿Cuáles son las tres formas para vincular CSS con HTML?

Las tres formas de vincular CSS con HTML son:
Etiqueta Style
En nuestro HTML y dentro de la etiqueta "head" utilizaremos la etiqueta "style" es decir que los CSS se aplican al MISMO docuemento.



Atributo Style
En nuestro HTML aplicaremos estilos directamente a la etiqueta. Recordemos que los atributos con características de un etiqueta.



Utilizando un Archivo Externo
esta opcion nos permite vincular un archivo HTML con un archivo CSS. Los archivos CSS tienen extensión .css. Suelen llamarse estilos.css o style.css
